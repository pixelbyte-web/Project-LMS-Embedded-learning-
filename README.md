# LMS Backend

### Crucial Commands

- **Install Dependencies:**
  ```bash
  npm install
  ```

- **Run Server (Backend):**
  ```bash
  npm start
  ```

- **Run UI (Frontend):**
  ```bash
  npm run dev
  ```

- **Seed Dummy Data:**
  ```bash
  node seed.js
  ```

### Requirements
You must create a `.env` file with `PORT`, `MONGODB_URI`, and `JWT_SECRET` for the server to run.
